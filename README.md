# Student Management System

## Description

Student Management System is a Python-based project that helps to manage student records easily.  
It is created using Object-Oriented Programming (OOP) concepts and provides a simple menu-driven system to perform different operations on student data.

## Features

- Add new student
- View all students
- Search student by ID
- Update student marks
- Delete student record
- Display student with highest marks
- Exit the program

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## Project Classes

### Student Class
This class stores student details like:
- Student ID
- Name
- Age
- Marks

### Student Management System Class
This class manages all student operations such as adding, searching, updating, deleting, and displaying student records.
